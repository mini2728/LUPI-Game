# LUPI-Game（最小正整數遊戲）

多人版的最小正整數遊戲（LUPI, Lowest Unique Positive Integer）。
支援
**主控端控制流程**、**玩家手機加入**、**多輪淘汰**，適合聚會、課堂與活動使用。

------------------------------------------------------------------------

## 🚀 安裝與啟動

### 1. 安裝套件（第一次）

``` bash
npm install express socket.io
```

### 2. 啟動伺服器

``` bash
node server.js
```

伺服器預設埠口：**3000**

------------------------------------------------------------------------

## 🖥 主控端（主持人操作）

### 1. 開啟主控介面

    http://你的IP:3000/main

⚠ 主控端電腦與玩家手機需在同一區網（同 Wi-Fi 或同熱點）。

------------------------------------------------------------------------

### 2. 設定可遊玩人數

輸入數字後按下 **「設定人數」**。
設定完成後，才允許玩家加入。

------------------------------------------------------------------------

### 3. 主控登入

-   預設管理密碼：`1234`
-   輸入後按下 **「登入主控」**

登入後可看到：

-   玩家列表\
-   遊戲狀態\
-   歷史紀錄\
-   控制按鈕（開始遊戲 / 開始統計 / 下一輪 / 重置遊戲）

------------------------------------------------------------------------

### 4. 玩家加入狀態

主控畫面會顯示：

    已加入 X / Y 人

人數達到後即可開始。

------------------------------------------------------------------------

### 5. 主控遊戲流程

1.  按 **「開始遊戲 / 下一輪」** → 玩家開始可選數字
2.  等玩家選完
3.  按 **「開始統計」** → 結算本輪
4.  如需繼續 → 再按 **「開始遊戲 / 下一輪」**

> 📝 本版本規則為：
> **唯一的最小正整數者獲勝-當觀眾**（其餘玩家進入下一輪）。

淘汰者後續只能觀戰。

------------------------------------------------------------------------

## 📱 玩家端（手機操作）

玩家以手機瀏覽器開啟：

    http://你的IP:3000/player

### 玩家流程

1.  輸入暱稱 → 按送出
    -   暱稱會保存在 localStorage
2.  等待主控開始遊戲
3.  選擇數字並送出
4.  若被淘汰 → 後續變成觀眾模式

------------------------------------------------------------------------

## 🏆 遊戲規則：LUPI（Lowest Unique Positive Integer）

每位玩家於每輪選擇 **1～9 之間任一正整數**。

-   若某數字為該輪 **唯一出現且最小** → 選該數字者 **淘汰**
-   若無唯一最小值 → 全員安全
-   最後存活者即為 **冠軍**

------------------------------------------------------------------------

## 🔧 系統特色

-   ✔ 主控端完全掌控遊戲節奏
-   ✔ 玩家端即時同步
-   ✔ 多輪淘汰制
-   ✔ 支援手機加入
-   ✔ 主控可強制重置遊戲
-   ✔ 重置時會清除所有玩家資料與連線

------------------------------------------------------------------------

## 📝 注意事項

-   主控與玩家需在同一區網
-   若要讓外網玩家加入，可設定 Port Forwarding 或使用內網穿透工具
-   多人同時加入時，建議使用穩定的 Wi-Fi
